# Dhan-Super-Order-Points-Point-Addon-Extension

### 🔵 Dhan Super Order Points (Point Addon)
## dhan-Super-Order-Points-extension.zip

A Chrome extension for Dhan that automatically fills the Target and Stop Loss price fields in Dhan's order form based on point values you set — so you don't have to manually calculate prices.

### Key features:

* Shows live price from the Dhan page in a floating panel
* You pick Target points and SL points (e.g., 50 pts target, 30 pts SL)
* Choose BUY or SELL — it calculates the correct Target/SL prices automatically
* When activated, it injects the calculated values directly into Dhan's order input fields (vt_target, vt_stoploss)
* Auto-update mode keeps recalculating as the live price moves
* Unlike the paper trade extensions, this works with real orders — it just auto-fills the form fields to save time
